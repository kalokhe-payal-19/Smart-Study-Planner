# 📚 Smart Study Planner

A clean and interactive productivity web application designed to help students manage their study routine, track tasks, maintain streaks, and stay organized efficiently.

---

## 🚀 Features

✅ Task Management System  
✅ Study Streak Tracker  
✅ Calendar Section  
✅ User Profile Page  
✅ Clean & Responsive UI  
✅ Easy Navigation Between Pages  

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
